
## 🙋‍♀️ 프로젝트 주제 및 진행 기간
* 2021-1학기 분산 프로그래밍 1 강의 학기말 프로젝트

* 2021.03 ~ 2021.06

## 🎈 프로젝트 내용

* 보험사 RFP 문서를 분석하여, 기능 요구 사항을 도출한다. 
* 각각의 요구 사항은  하나의 use case로써, 시나리오를 갖는다.
* use case를 바탕으로 class diagram으로 전체 시스템 구조를 설계한다.
* 설계도를 기반으로 하여 TUI 방식의 보험사 시스템을 구현한다.
